# Playwright Data-Driven Tests

Data-driven Playwright automation framework built with TypeScript for validating task management workflows and UI elements.

## Tech Stack
- Playwright
- TypeScript
- Node.js

## Features
- Data-driven test structure
- Reusable login automation
- Scalable test design
- UI validation with Playwright assertions

## Run Tests

Install dependencies:

npm install

Run tests:

npx playwright test

Run headed mode:

npx playwright test --headed
